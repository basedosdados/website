Você já pensou em transformar dados em mapas interativos e análises geoespaciais? O QGIS, um Sistema de Informação Geográfica (SIG) de código aberto, gratuito e colaborativo, é a ferramenta ideal para isso. E a boa notícia é que preparamos um curso completo e totalmente gratuito para você iniciar ou aprimorar sua jornada nesse universo!

Este curso foi cuidadosamente desenvolvido para te guiar desde os conceitos fundamentais da cartografia até a criação de animações complexas e a aplicação do QGIS em reportagens investigativas. Com 6 aulas práticas e didáticas, você aprenderá a manipular dados, gerar visualizações impactantes e extrair insights valiosos.

## Aula 1: Começando a usar o QGIS

Nesta primeira aula, você será introduzido aos conceitos essenciais de cartografia, entenderá a interface do QGIS e o que é um SIG, aprenderá sobre os diferentes tipos de dados geoespaciais e, finalmente, criará o seu primeiro mapa. É o ponto de partida perfeito para quem nunca teve contato com a ferramenta.

## Aula 2: Dados sobre Queimadas

Aprofunde-se em um tema de grande relevância: a análise de dados sobre queimadas. Esta aula aborda o monitoramento do INPE, discute as limitações e possibilidades na análise desses dados e explora os marcos legais da legislação ambiental relacionados à análise de dados geoespaciais. Uma oportunidade para entender como o QGIS pode ser aplicado em questões socioambientais críticas.

Descubra como integrar o vasto acervo da Base dos Dados com o QGIS. Você aprenderá a importar dados de queimadas diretamente da Base dos Dados e a manipulá-los dentro do QGIS, abrindo um leque de possibilidades para suas análises com dados públicos e tratados.

## Aula 4: Criando mapas de Calor no QGIS

Os mapas de calor são ferramentas visuais poderosas para identificar padrões e concentrações. Nesta aula, você aprenderá os conceitos essenciais para a criação desses mapas e, em seguida, colocará a mão na massa para criar seus próprios mapas de calor no QGIS, transformando dados brutos em visualizações intuitivas.
