Os dados BD Pro podem ser acessados de diferentes formas:

* Diretamente no Google BigQuery, utilizando SQL para fazer consultas
* Por meio dos pacotes da BD para Python, R e Stata
* Através de ferramentas de BI, como Power BI, Tableau, Qlik entre outras.

Em todos os casos você precisará criar um projeto no BigQuery com o seu e-mail cadastrado na BD Pro. 

Para mais detalhes, acesse nossa [documentação](https://basedosdados.org/docs/home)
