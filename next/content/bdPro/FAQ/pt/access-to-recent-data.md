Os dados mais recentes estão disponíveis exclusivamente para assinantes BD Pro. Para ter acesso, basta [assinar o serviço](#pricing) — qualquer pessoa ou organização pode assinar sem restrições. Assim que o pagamento é concluído, o acesso é liberado automaticamente e você já pode utilizar os dados atualizados na plataforma.
